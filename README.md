# Генерация содержательного описания изображений открыток и плакатов

## Обзор

Данный проект реализует прототип системы автоматической генерации структурированных и семантически насыщенных описаний изображений (открыток и плакатов). Основная цель — улучшение поиска и доступа к визуальным материалам в цифровых архивах и библиотеках.

Система объединяет методы компьютерного зрения и обработки естественного языка для преобразования визуального контента в человекочитаемые описания и поисковые метаданные.

---

## Требования

Установка зависимостей:

```bash
pip install -r requirements.txt
```

## Архитектурный подход

Система предполагает реализацию в виде модульного vision-language пайплайна, разделяющего этапы восприятия, семантической интерпретации и генерации текста:

```
  - Генерация описания (пока - BLIP)
  - Семантическая классификация (пока - CLIP, zero-shot)
  - Сборка описания
  - Структурированный результат
```

Такой подход позволяет отказаться от жёстко заданных правил и использовать предобученные мультимодальные модели для обобщения на различные типы изображений.

## Предположительные основные компоненты

1. Генератор описания (Caption Generator)
- Модель: BLIP
- Задача: извлечение базового описания сцены
- Выход: описание на английском + перевод на русский
2. Извлечение метаданных (Metadata Extractor)
- Модель: CLIP (zero-shot классификация)
- Извлекаемые признаки:
  - тип изображения (открытка, плакат и др.)
  - стиль (винтаж, иллюстрация и др.)
  - тематика (праздничная, романтическая и др.)
  - настроение (радостное, ностальгическое и др.)
3. Семантический слой (Theme Inference)
- Обеспечивает согласованную структуру семантических признаков
- Использует результаты CLIP 
4. Генерация архивного описания (Description Builder)
Формирует:
- развернутое архивное описание
- поисковую строку (ключевые слова)
5. Модуль оценки 
Разделение метрик на два уровня:
- Лексические метрики: BLEU, ROUGE, METEOR
- Семантическая метрика: BERTScore

## Использование

Запуск пайплайна для одного изображения

```bash
PYTHONPATH=src python src/run_demo.py --image data/postcard.jpg

```

Пример результата:

```bash
{
  "caption": {
    "en": "...",
    "ru": "..."
  },
  "metadata": {

    "style": "...",
    "theme": "...",
    "mood": "..."
  },
  "archive_description": "...",
  "search_text": "..."
}

```

## Примечания 
- В текущей версии используются предобученные модели, обучение не требуется.
- OCR (распознования текста) не включён в MVP из-за низкой надёжности извлечения текста через captioning.
- Архитектура спроектирована как расширяемая (возможна интеграция OCR, fine-tuning, специализированных классификаторов).
